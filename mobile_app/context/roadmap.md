# Roadmap

- Phase 1: Foundation
  - Project bootstrap, navigation, auth, theme, i18n
- Phase 2: Core Customer Experience
  - Browsing (taxis/routes/tours), booking flow, booking tracking
- Phase 3: Payments & Support
  - Payment method selection, proof upload, reviews, complaints, tourist assistance, WhatsApp deep links
- Phase 4: Driver Experience
  - Driver dashboard, trip status updates, earnings, review visibility
- Phase 5: Polish & Release
  - Offline caching, notifications, QA, performance, store submission
